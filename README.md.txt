# Student Result Management System

A console-based application developed using Core Java, JDBC, and MySQL for managing student academic records.

## Features
- Add Student Records
- View Student Details
- Add Subject-wise Marks
- Calculate Average and Grade
- Delete Student Records
- View All Students

## Technologies Used
- Core Java
- JDBC
- MySQL
- Collections Framework
- Java 8 Streams

## Concepts Demonstrated
- OOP (Encapsulation, Abstraction)
- Exception Handling
- JDBC Connectivity
- ArrayList Collections
- PreparedStatement
